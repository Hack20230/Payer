# Payer

## 最强子域名挖掘机一分钟上万数据

mac、windows、linux均可使用，一个字 快！

使用说明

./Payer -help 查看帮助

Payer.exe -help 查看帮助

```
-d 扫描需要爆破的二级域名、三级域名、四...

-t 添加线程

-x 不想显示的size值 -x 100,200,300同时使用

-xmax 不想显示的最大size值 

-xmin 不想显示的最小size 

-xmax -xmin 可以同时匹配

-p 代理,如 http://username:password@ip:port, http://ip:port
 
```
